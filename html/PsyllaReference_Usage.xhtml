2.1. Command Line Interface

2.1.1. Synopsis

psylla [option…] [--] [program [argument…]]

2.1.2. Command Line Parameters

options
--help, --usage, -h, -?
Show options summary.
--version, -V
Show the version of the interpreter.
--config template, --cfg template
Print to standard output the value of the named configuration properties. The template can contain shell-like metacharacters * and ?.
--eval, -e expression
Evaluate an expression consisting of Psylla commands.
--classpath, --cp path
Additional extensions class path including directories and jar archives.
--librarypath, --lp, -I path
Additional modules search path.
--console-encoding, -C encoding
Set the encoding of the console.
--locale, -L locale
Set the default locale.
--random-seed, -S seed
Initialize the pseudorandom numbers generator with seed.
program
The name of a file containing Psylla program. The program is read from the standard input when the hyphen is specified. If the program is not specified, run REPL (read—evaluate—print loop).
argument
Argument passed to Psylla program.

2.2. Java Scripting API

The Psylla provides support for Java Scripting API (JSR 223). The jrunscript shell demonstrates this feature: 

% jrunscript -cp target/lib/psylla.jar -l psylla -e '1 1 1 7 { mul } for ?'
5040
